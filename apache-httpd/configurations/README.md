### .HTACCESS   

<p> .htaccess files provide a way to make configuration changes on a per-directory basis </p>

### directory based configuration can be this 

```
/.htaccess
/www/.htaccess
/www/htdocs/.htaccess
/www/htdocs/example/.htaccess
```
